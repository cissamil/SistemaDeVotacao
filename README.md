# Sistema De Votacao

Atividade em React + Vite para simular um sistema de votacao simples

## Regras da atividade

- Cada candidato comeca com 0 votos.
- Ao clicar em `Votar`, incrementa apenas 1 voto para o candidato.
- Ao finalizar, deve mostrar o resultado final e o vencedor.

## Como executar

```bash
npm install
npm run dev
```
